# G2 Autoria e Design WEB
 
** CUIDADOS AO EDITARO O PROJETO **

- Sempre avisar no grupo quando está editando algo no projeto principal.
- Se possível, realizar um teste com uma cópia fora da pasta GIT e somente após o término/validação editar o arquivo do repositório.
- Tentar manter o mesmo padrão de layout, caso queira realizar mudanças favor informar os outros no grupo.
- Ideias são sempre bem vindas, e se compartilhadas com o grupo, podem ser melhoradas.

Obrigado pela atenção!
